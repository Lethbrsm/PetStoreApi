Stephen Lethbridge // Pet Store Web API Build Instructions

1. Download the .zip file of the PetStore project from https://github.com/Lethbrsm/PetStoreApi
2. Extract the zip file to a new folder.
3. Open Visual Studio 2017.
4. Open the PetStore.sln file in VS from the extracted zip.
5. Right click on the PetStore Solution in the Solution Explorer and choose Manage NuGet Packages.
6. Click on the restore button to restore the packages utilized by the project. If these are not installed already, 
download the following packages: bootstrap, jQuery, LiteDB, and json.
7. Repeat step 6 for the PetStoreTests solution if necessary. 
8. Choose the Release build and run the Web API on a web browser.
